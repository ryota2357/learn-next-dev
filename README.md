# learn-next-dev

> Learning project of Next.js.
> - TypeScript
> - App Router
> - MUI (with Emotion)
> - SWR, Axios
> - Recoil

## Usage

Start develop server:

```sh
npm run dev
```

Lint and format code:

```sh
npm run lint
npm run format

# or dry run
npm run lint:check
npm run format:check
```
